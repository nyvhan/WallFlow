Fonds d'écran animés sur Windows

Suis ces 3 étapes pour utiliser tes vidéos WallFlow comme fond d'écran avec Lively Wallpaper.

1

Télécharger

Choisis un fond sur WallFlow, ouvre sa page détail puis clique sur Télécharger. Le fichier .mp4 sera enregistré sur ton PC.

Voir la collection WallFlow
2

Installer Lively Wallpaper

Sur Windows, utilise Lively Wallpaper pour afficher des vidéos en fond d'écran. Télécharge-le gratuitement depuis le Microsoft Store.

Installer Lively Wallpaper (Microsoft Store)
3

Appliquer

Ouvre Lively Wallpaper, puis glisse-dépose ton fichier .mp4 dans la fenêtre (ou ajoute-le via le bouton +). Le fond d'écran est appliqué aussitôt.

.mp4 → Écran
